This repository is created as part of my 30-Day Verilog HDL Challenge, where I will be implementing and verifying digital designs ranging from basic combinational circuits to advanced RTL modules.
The objective of this challenge is to strengthen my understanding of digital logic design, Verilog programming, RTL coding practices, and hardware verification concepts that are essential for VLSI and ASIC design.
Each design includes:
- Verilog RTL code
- Testbench for verification
- Simulation results/waveforms
- Concepts learned during implementation
The journey starts with fundamental circuits like Half Adder and Full Adder and gradually progresses towards complex designs such as FSMs, FIFOs, UART, and processor-related modules.
